# QaaS.Common.Generators

Contains commonly used generators that can be added as a plugin to a `QaaS` project by adding the package to the dotnet project.

## Usage

Add the following with a version of your choice to ur `.csproj` file

```xml
<ItemGroup>
      <PackageReference Include="QaaS.Common.Generators" Version="x.x.x" />
</ItemGroup>
```
