# Stacking Table View

| Property Path                                   | Type                                           | Required   | Default   | Description                                                                                                                                                                                                                 |
|:------------------------------------------------|:-----------------------------------------------|:-----------|:----------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `GeneratorConfiguration.ItemsPerGenerator`      | array (length >= 1)                            | &#10004    |           | The number of items to generate out of the given data sources in turn                                                                                                                                                       |
| `GeneratorConfiguration.Count`                  | integer (value >= 0.0 / value <= 2147483647.0) | &#10006    |           | The number of items to generate out of the given data sources (If count is bigger than available number of items an exception will be thrown), if no value is given generates the number of items in the given data sources |
| `GeneratorConfiguration.LoopFinishedGenerators` | boolean                                        | &#10006    | False     | When true, generators will restart when there are no more items in them                                                                                                                                                     |
| `GeneratorConfiguration`                        | object                                         | &#10004    |           | Supports generating data from multiple DataSources by configured items per generator. `DataSources`: Used. `SessionData`: Passed to used DataSources.                                                                       |
