# FromFileSystem

Retrieves data from a configured path to a directory in the file system.

!!! info "Data Structure"
    === ":octicons-file-code-16: `Data`"
        ```yaml
        Body: <byte[]>
        MetaData:
            Storage:
              Key: <string> # The identifier of the item, decided by the `StorageMetaData` configuration property
        ```

## DataSource Parameters

* **Generator Name** - `FromFileSystem`.
* **DataSources** - Not Used.
* **SessionData** - Not Used.
