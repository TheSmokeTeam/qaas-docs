# S3

## Available Probes

- `EmptyS3Bucket`
- `DeleteS3Bucket`

## Configuration Fields

Both S3 probes build on the shared bucket configuration:

| Field | Required | Description |
|---|---|---|
| `StorageBucket` | yes | Bucket name. |
| `ServiceURL` | yes | S3-compatible service URL. |
| `AccessKey` | yes | Access key. |
| `SecretKey` | yes | Secret key. |
| `ForcePathStyle` | no | Use path-style addressing. Default: `true`. |

`EmptyS3Bucket` also supports:

| Field | Required | Description |
|---|---|---|
| `Prefix` | no | Delete only objects under this prefix. Default: empty string. |

Use `EmptyS3Bucket` when the bucket should remain in place. Use `DeleteS3Bucket` only when the bucket itself is disposable in the target environment.
