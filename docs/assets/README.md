# Assets Directory

Place your documentation assets here:
- `logo.png` - Site logo (recommended: 256x256 px)
- `favicon.png` - Browser favicon (recommended: 32x32 px)
- Any other images or static files referenced in documentation

## Recommended Logo Specs

- Format: PNG with transparency
- Size: 256x256 pixels for logo
- Size: 32x32 pixels for favicon
