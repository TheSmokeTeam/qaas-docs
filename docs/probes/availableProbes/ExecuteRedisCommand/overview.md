# ExecuteRedisCommand

Executes one Redis command with optional arguments against the selected Redis database.

## Probe Parameters

- **Probe Name** - `ExecuteRedisCommand`.
- **ProbeConfiguration** - `RedisExecuteCommandConfig`.
- **DataSources** - Not used.
- **SessionData** - Not used.
- **Execution stage** - Typically used in setup or cleanup stages around Runner sessions.
