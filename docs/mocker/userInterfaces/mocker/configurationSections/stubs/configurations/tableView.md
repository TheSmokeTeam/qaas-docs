# Stubs Configurations Table View

| Property Path                                                  | Type                                                                                           | Required   | Default   | Description                                                                                                                                                  |
|:---------------------------------------------------------------|:-----------------------------------------------------------------------------------------------|:-----------|:----------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `Stubs`                                                        | `array`                                                                                        | &#10006    |           | List of transaction stubs that can be used for server actions. They provide processing functionality to exercise transaction data.                           |
| `Stubs[]`                                                      | `object`                                                                                       | &#10006    |           |                                                                                                                                                              |
| `Stubs[].Processor`                                            | `string`                                                                                       | &#10004    |           | The processor hook to use. This can be a full type name or a unique processor class name.                                                                   |
| `Stubs[].Name`                                                 | `string`                                                                                       | &#10004    |           | Name of the stub to reference it by (must be unique).                                                                                                        |
| `Stubs[].DataSourceNames`                                      | `array`                                                                                        | &#10006    |           | Names of data sources to pass to this stub. Those data sources do not have to be defined before this stub in the file.                                      |
| `Stubs[].DataSourceNames[]`                                    | `string`                                                                                       | &#10006    |           |                                                                                                                                                              |
| `Stubs[].ResponseBodySerialization`                            | `object`                                                                                       | &#10006    |           | Serialization settings to apply to the response body after the processor returns.                                                                            |
| `Stubs[].ResponseBodySerialization.Serializer`                 | `enum` [`Binary` / `Json` / `MessagePack` / `Xml` / `Yaml` / `ProtobufMessage` / `XmlElement`] | &#10006    |           | Serializer type from `QaaS.Framework.Serialization`. Null means no serialization is applied.                                                                 |
| `Stubs[].RequestBodyDeserialization`                           | `object`                                                                                       | &#10006    |           | Deserialization settings to apply to the request body before the processor runs.                                                                             |
| `Stubs[].RequestBodyDeserialization.Deserializer`              | `enum` [`Binary` / `Json` / `MessagePack` / `Xml` / `Yaml` / `ProtobufMessage` / `XmlElement`] | &#10006    |           | Deserializer type from `QaaS.Framework.Serialization`. Null means no deserialization is applied.                                                             |
| `Stubs[].RequestBodyDeserialization.SpecificType`              | `object`                                                                                       | &#10006    |           | Configuration for deserializing into a specific C# type. When omitted, the default type for the selected deserializer is used.                              |
| `Stubs[].RequestBodyDeserialization.SpecificType.TypeFullName` | `string`                                                                                       | &#10004    |           | The full name (including path) of the type                                                                                                                   |
| `Stubs[].RequestBodyDeserialization.SpecificType.AssemblyName` | `string`                                                                                       | &#10006    |           | The assembly that contains the type. If omitted, the runtime falls back to the entry assembly.                                                               |
| `Stubs[].ProcessorSpecificConfiguration`                       | `object`                                                                                       | &#10006    |           | Implementation-specific configuration bound into the selected processor.                                                                                      |
