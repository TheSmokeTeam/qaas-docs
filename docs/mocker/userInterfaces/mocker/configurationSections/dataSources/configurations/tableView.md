# DataSources Configurations Table View

| Property Path                                         | Type                                                                                           | Required   | Default   | Description                                                                                                                                                                     |
|:------------------------------------------------------|:-----------------------------------------------------------------------------------------------|:-----------|:----------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `DataSources`                                         | `array`                                                                                        | &#10006    |           | List of data sources that can be used by stubs, other data sources, or socket broadcast actions.                                                                                |
| `DataSources[]`                                       | `object`                                                                                       | &#10006    |           |                                                                                                                                                                                 |
| `DataSources[].Generator`                             | `string`                                                                                       | &#10004    |           | The name of the generator to use                                                                                                                                                |
| `DataSources[].Name`                                  | `string`                                                                                       | &#10004    |           | Name of the data source to reference it by (must be unique)                                                                                                                     |
| `DataSources[].Lazy`                                  | `boolean`                                                                                      | &#10006    | False     | True to iterate over data lazily                                                                                                                                                |
| `DataSources[].DataSourceNames`                       | `array`                                                                                        | &#10006    |           | Names of data sources to pass to this data source. Those data sources do not have to be defined before this item in the file.                                                   |
| `DataSources[].DataSourceNames[]`                     | `string`                                                                                       | &#10006    |           |                                                                                                                                                                                 |
| `DataSources[].Deserialize`                           | `object`                                                                                       | &#10006    |           | Deserialize to use on the generated data                                                                                                                                        |
| `DataSources[].Deserialize.Deserializer`              | `enum` [`Binary` / `Json` / `MessagePack` / `Xml` / `Yaml` / `ProtobufMessage` / `XmlElement`] | &#10006    |           | The deserializer type to use. Null means no deserialization is applied. Options come from `QaaS.Framework.Serialization`.                                                      |
| `DataSources[].Deserialize.SpecificType`              | `object`                                                                                       | &#10006    |           | Configuration for deserializing into a specific C# type. When omitted, the default type for the selected deserializer is used.                                                  |
| `DataSources[].Deserialize.SpecificType.TypeFullName` | `string`                                                                                       | &#10004    |           | The full name (including path) of the type                                                                                                                                      |
| `DataSources[].Deserialize.SpecificType.AssemblyName` | `string`                                                                                       | &#10006    |           | The name of the assembly the type is located in. If no value is given, the runtime falls back to the entry assembly.                                                            |
| `DataSources[].Serialize`                             | `object`                                                                                       | &#10006    |           | Serialize to use on the generated data                                                                                                                                          |
| `DataSources[].Serialize.Serializer`                  | `enum` [`Binary` / `Json` / `MessagePack` / `Xml` / `Yaml` / `ProtobufMessage` / `XmlElement`] | &#10006    |           | The serializer type to use. Null means no serialization is applied. Options come from `QaaS.Framework.Serialization`.                                                          |
| `DataSources[].GeneratorSpecificConfiguration`        | `object`                                                                                       | &#10006    |           | Implementation-specific configuration bound into the selected generator.                                                                                                         |
