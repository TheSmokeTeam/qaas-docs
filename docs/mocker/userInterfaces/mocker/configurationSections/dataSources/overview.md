# DataSources

`DataSources` define reusable collections of data generated by a named `IGenerator` hook.

In QaaS.Mocker, data sources are commonly used by:

- `Stubs`, through `DataSourceNames`
- socket broadcast actions, through `Action.DataSourceName`
- other data sources, through `DataSourceNames` and `DataSourcePatterns`

The current configuration model uses `GeneratorConfiguration` for hook-specific settings. `GeneratorSpecificConfiguration` is the older name and should not be used in new files.
