# QaaS.Framework.Serialization

Contains serialization/deserialization functionality
This package is installed as part of the [QaaS.Framework.SDK](sdk.md) Package.

## Supported Formats

| Format      | Serializer | Deserializer |
|-------------|------------|--------------|
| Binary      | ✔          | ✔            |
| Json        | ✔          | ✔            |
| Xml         | ✔          | ✔            |
| XmlElement  | ✔          | ✔            |
| MessagePack | ✔          | ✔            |
| Yaml        | ✔          | ✔            |
| Protobuf    | ✔          | ✔            |
