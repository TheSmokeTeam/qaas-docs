# Plugins

A QaaS Plugin is a NuGet package containing **QaaS hooks** (generators, assertions, probes, or processors). Plugins must reference `QaaS.Framework.SDK`.

---

## Built-in Plugins

Examples of built-in QaaS Plugins include:

- [QaaS.Common.Generators](REDA/v3/)  

- [QaaS.Common.Assertions](REDA/v3/)  

- [QaaS.Common.Probes](REDA)
