# Plugins

> TL;DR — QaaS Plugins are C# NuGet packages that contain QaaS hooks, extensibility points that integrate with QaaS.Framework. To qualify as a QaaS Plugin, a package must use QaaS.Framework.SDK.

QaaS Plugins are C# NuGet packages that contain QaaS hooks, extensibility points that integrate with [QaaS.Framework](../../framework/index.md). To qualify as a QaaS Plugin, a package must use [QaaS.Framework.SDK](../../framework/projects/sdk.md).
