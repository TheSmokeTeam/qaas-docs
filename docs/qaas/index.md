# QaaS Runner

**QaaS.Runner** is the test orchestrator of the QaaS ecosystem. It uses YAML- or code-based configuration to execute multi-stage communication sessions against backend services, persist results, evaluate assertions, and generate [Allure]({{ links.allure_docs }}) test reports.
