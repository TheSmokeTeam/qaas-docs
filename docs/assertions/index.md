# QaaS.Common.Assertions

Contains commonly used assertions that can be added as a plugin to a `QaaS` project by adding the package to the dotnet project.

## Usage

Add the following with a version of your choice to ur `.csproj` file

```xml
<ItemGroup>
      <PackageReference Include="QaaS.Common.Assertions" Version="x.x.x" />
</ItemGroup>
```
